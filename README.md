# emetoverman.github.io

Jekyll-based GitHub Pages site for [EmetOverman](https://github.com/EmetOverman),
adapted from the
[brutalist-blog](https://github.com/andrewhwanpark/brutalist-blog) theme by
Andrew Hwan Park.

## Local development

```sh
bundle install
bundle exec jekyll serve
```

## Deployment

Push to `main`. The workflow in
`.github/workflows/pages.yml` builds and deploys the site to GitHub Pages.
